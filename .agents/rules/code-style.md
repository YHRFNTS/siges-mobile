Ok este es un proyecto de android studio que usa la api de SIGES. Es una aplicación móvil con 2 tipos de usuario que siguen 2 flujos distiotos: Los aplicantes (Student y Institutional Staff) que pueden hacer reservaciones, y el Admin, que puede ver de manera rápida las reservaciones que tiene pendiente por aceptar o rechazar. Porfavor analiza la estructura que tengo del proyecto para que todo lo que te pida tenga la misma estructura MVVM, que sea altamente componentizable y con las mejores prácticas. Usa MaterialTheme y los custom SigesTheme que tengo para colores, formas y tipografía. Cuando se pueda, acompaña nuevas funcionalidades con test para validar que funciona
